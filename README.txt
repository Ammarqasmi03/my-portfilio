Portfolio Website
=================

Project: Personal portfolio website showcasing projects, skills, and contact information.

Contents
--------
- index.html        : Main landing page
- css/              : Stylesheets
- js/               : JavaScript files
- assets/           : Images and other media

Getting Started
---------------
1. Clone or copy the project folder to your local machine.
2. Open index.html in a browser to view the site locally.

Development
-----------
- Edit HTML in index.html.
- Modify styles in css/ (e.g., styles.css).
- Update interactivity in js/ scripts.

Build & Deployment
------------------
- This is a static site. Deploy to GitHub Pages, Netlify, Vercel, or any static host.
- To deploy on GitHub Pages: push the repository to GitHub and enable Pages in repository settings.

Contributing
